## Dependencies 

x11, Xinerama, xcb, freetype, notofont-cjk

## Make dwm

`make && sudo make clean install`

### Patches

All of the patches that I applied are in the patch folder but here are the names of them

- vanitygaps
- hide vacant tags
- attachbelow
- autostart
- pertag
